## General Information


 This is the demo files for configuring a ST Servo with an esp32. The goal of this project was to get the esp32 reading and controling the Servo Motor. I developed had the following key features: 
- [x] Ping Servo
- [x] Set Servo ID
- [x] Set Servo Mode 
- [x] Write to servo
- [x] Get feedback

## Project File Structure

The following is a breakdown of the different folders and the files contained in them:

1. **Examples**
    - Contains copy of differenct test files. 

  
    &nbsp;
2. **GH PAGES**
    - Files used for Github Pages and readme text file. 

    &nbsp;
3. **Arduino Library**
   - Contains both the original and modified libary for the STServo. the Modified libary should be same files as found in the example files. Add the libary to you Arduino IDE. from the arduino IDE launch an example file and  upload to the microcontroler. 
    

> Note: Files are not optimized and could contain unused code, references, documents.
